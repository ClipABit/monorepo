# ClipABit Server


### How to run
1. Open a terminal and navigate to `/backend`.
2. Create a new python venv
3. Install all dependencies
4. Open `modal.com` on your browser and sign in with the ClipABit gmail (reach out to TPM for credentials)
4. run `modal token new`. A browser should open to confirm the account verification
5. run `modal serve main.py` to start the container in the cloud.
